Dibujemos

Canvas es una de las herramientas más utlies para el diseño web debido a que estos ayudan a la creaciones de imagenes o gráficos de manera sencilla sin tener que complicarse de diferentes maneras, ya sea con el css o SVG, en este caso hemos usado canvas para poder crear una herramienta simple usando solo dos cosas, Canvas y también JavaScript, a está aplicación le hemos llamado Dibujemos, en la cual puedes dibujar lo que tu quieras en este sitio web.
