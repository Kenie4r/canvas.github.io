EL PRINCIPAL OBJETIVO DE ESTA PÁGINA

El objetivo de este sitio es el conocer sobre una de las herramientas más importantes en el desarrollo web, como lo es Canvas, en la cual se puede dibujar , crear imagenes y también algo muy importante como lo es gráficas, esto solo usando JavaScript y HTML, sin el uso de css debido a que este crea más complicaciones, en este caso hemos creado una pequeña aplicación en la cual podremos dibujar en este sitio web (haciendo uso de Canvas y JavaScript).

¿Qué es Canvas?

La definición de la palabra canvas proviene de forma en latín la cual significa lienzo; En HTML este no pierde el significado, siendo un lienzo donde se puede dibujar o pintar lo que los diseñasdores quierán, para el uso de este es necesario el uso de scripts, los cuales se refieren al uso de JS, para poder dibujar en estos lienzos
